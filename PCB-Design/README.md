# PCB Design

This folder contains everything related to the two PCBs of the OSSM ALT Edition (the main PCB and the brake chopper), organized as follows:

| Folder | Contents |
| ------ | -------- |
| [Schematic and Design](Schematic%20and%20Design/) | The schematics of both PCBs and their STEP files |
| [Production Files](Production%20Files/) | A manual on ordering, plus all the files needed to produce the PCBs yourself (Gerbers, BOM, CPL) |
| [Kicad Files](Kicad%20Files/) | The KiCad project files, if you wish to edit things |
